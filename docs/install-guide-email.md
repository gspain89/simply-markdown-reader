## Simply Markdown Reader 설치 안내

안녕하세요.

마크다운(.md) 파일을 편하게 읽을 수 있는 뷰어를 공유드립니다.
기존에는 .md 파일을 열면 텍스트 편집기에서 코드처럼 보이거나, 별도 프로그램이 없어서 불편하셨을 텐데요.
이 프로그램을 설치하시면 .md 파일을 더블클릭만으로 깔끔하게 읽으실 수 있습니다.

---

### 1. 설치

첨부된 **SimplyMarkdownReader-1.0.3.dmg** 파일을 열어주세요.

1. DMG 파일을 더블클릭합니다
2. 열리는 창에서 **Simply Markdown Reader** 아이콘을 **Applications** 폴더로 드래그합니다
3. 설치 끝입니다

#### 처음 실행 시 보안 경고 해결 방법

처음 앱을 열면 **"Simply Markdown Reader을(를) 열지 않음"** 경고가 나타납니다.
Apple 공식 스토어가 아닌 외부 앱이라 뜨는 정상적인 절차이며, 아래 단계로 해결됩니다. (최초 1회만)

1. 경고 창에서 **"완료"** 를 누릅니다 ("휴지통으로 이동"을 누르지 마세요)
2. **시스템 설정** 을 엽니다 (좌측 상단 Apple 로고 → 시스템 설정)
3. **개인정보 보호 및 보안** 을 클릭합니다
4. 화면 하단으로 스크롤하면 **"Simply Markdown Reader이(가) 차단되었습니다"** 메시지가 보입니다
5. **"그래도 열기"** 버튼을 클릭합니다
6. 비밀번호를 입력하면 앱이 정상 실행됩니다

이후부터는 경고 없이 바로 실행됩니다.

---

### 2. 기본 프로그램으로 지정 (중요)

이 설정을 해두시면 앞으로 모든 .md 파일을 더블클릭만으로 바로 열 수 있습니다.

1. 아무 **.md 파일**을 하나 찾아서 **우클릭** (또는 Control+클릭) 합니다
2. **정보 가져오기** 를 클릭합니다
3. 정보 창에서 **다음으로 열기** 항목을 찾습니다
4. 드롭다운 메뉴에서 **Simply Markdown Reader** 를 선택합니다
5. 바로 아래에 있는 **모두 변경...** 버튼을 반드시 클릭합니다
6. 확인 팝업이 뜨면 **계속** 을 누릅니다

**"모두 변경"을 누르지 않으면** 해당 파일 하나에만 적용되고, 다른 .md 파일은 여전히 기존 프로그램으로 열립니다. 반드시 눌러주세요.

---

### 3. 사용법

설치와 기본 프로그램 지정이 끝나면, .md 파일을 더블클릭하시면 됩니다.

- 왼쪽 **목차**에서 원하는 섹션으로 바로 이동할 수 있습니다
- **Cmd+F**로 문서 내 검색이 가능합니다
- 우측 상단 해/달 아이콘으로 **다크모드 전환**이 가능합니다
- **PDF 내보내기**: 상단 공유 아이콘 → Export as PDF
